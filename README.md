## Usage
   1. Clone this repository
      ```
      git clone http://github.com/postxml/postxml-plugin-boilerpate.git
      ```
   2. Fill `plugin.json`
   3. Run
      ```
      node plugin.js
      ```
   4. Install packages
      ```
      npm i
      ```
   5. Develop
      ```
      gulp watch
      ```
